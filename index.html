Principais Contribuições

Software Público Brasileiro

É uma plataforma colaborativa contendo ferramentas integradas visualmente, a nível de autenticação e busca integrada. Provendo serviços de rede social, listas de email e forge para controle de versão.

Link para a plataforma
Noosfero

Noosfero é uma plataforma web para redes de economia social e solidária com blog, e-Porfolios, CMS, RSS, discussão temática, agenda de eventos e inteligência coletiva para economia solidária no mesmo sistema.

Página oficial do Noosfero
Kuniri

Kuniri inspeciona o código fonte, extrai informações e leva para um output padronizadom dando flexibilidade para qualquer ferramenta aproveitar os metadados.

Página oficial do Kuniri
Colab

Permite a integração de ferramentas no que diz respeito a autenticação única, busca global e visual. Através de sinais é possível a comunicação entre as ferramentas integradas.

Repositório do Colab
Debian

Sistema operacional livre, sendo esse o maior projeto de software livre do mundo. Havendo contribuições no desenvolvimento de ferramentas de integração contínua, deploy de aplicações web e construção de pacotes. Além de membros dos time manterem pacotes dentro do sistema.

Página oficial do Debian
Mezuro

Mezuro é ma plataforma web livre para avaliação colaborativa de código fonte, você pode avaliar seu código fonte com os SCMs mais populares (como Git e SVN), você pode avaliar códigos em C, C++, Java, Ruby e Python.

Página Oficial do Mezuro
Analizo

Analizo é uma toolkit de ferramentas livre, multi-linguagem, de análise de código fonte extensível e visualização de métricas. Ele suporta a extração e cálculo de um bom número de métricas de código fonte, geração de gráficos de dependência e análise de evolução do software.

Página Oficial do Analizo
Radar Parlamentar

A tecnologia a serviço do cidadão / cidadã - É uma ferramenta para análise partidária das casas legislativas.

Página do Radar Parlamentar
GestorPsi

GestorPsi é uma ferramenta para gerenciar via Web as rotinas administrativas e técnicas e manter sua documentação regularizada segundo as normas éticas e profissionais do Conselho de Psicologia.

Página do GestorPsi
